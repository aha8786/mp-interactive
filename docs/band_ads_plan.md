## 1. 홈(`/`) — ServicesSection.tsx 전면 재작성

### 변경 전
- 서비스 카드 3개 (네이버밴드 / 구글광고 / SNS광고) 그리드
- 구글/SNS는 주석 처리된 상태

### 변경 후 구조
```
[① 섹션 헤더]
뱃지: "네이버 밴드 광고 대행"
제목: "국내 MAU 1,800만, 글로벌 1억 3천만의 플랫폼"
부제: "검증된 플랫폼, 데이터 기반 전략으로 실질 성과를 만듭니다"

[② 핵심 수치 4개 카드]
글로벌 가입자 1억 3,000만  |  국내 MAU 1,800만  |  소셜미디어 4위 28.6%  |  미국 MAU 710만
(CountUpNumber 애니메이션 적용)

[③ 비포/에프터]
[before.png 회색조]  →(화살표)→  [after.png 풀컬러]
"과거: 일괄 노출 방식"           "현재: 데이터 맞춤 전략"
하단 문구: "트렌드는 변합니다. 우리는 유연하게 대응합니다."

[④ AI/데이터 차별화 3포인트]
🔍 키워드 분석  |  🎯 정밀 타겟 설정  |  📈 광고 효율 개선

[⑤ CTA 버튼]
"네이버 밴드 광고 자세히 보기 →" → /services/naver-ads
```

### 디자인
- 배경: `#EDEAE1`
- 수치 색상: 그린 `#03C75A`
- before 이미지: `filter: grayscale(60%)` 적용
- after 이미지: 그린 보더 강조

## 2. 상세 페이지(`/services/naver-ads`) — 전면 재구성

### 변경 전 구조
```
① 히어로
② 세부 서비스 탭 (스폰서/피드/지역/커뮤니티) ← 내용 오류
③ FAQ
④ CTA
```

### 변경 후 구조
```
① 히어로 (문구 수정)
② 국내 MAU 1,800만 통계 섹션        ← 신규
③ 광고 게재 위치 3장 나란히          ← 신규 (탭 대체)
④ AI/데이터 차별화 섹션             ← 신규
⑤ FAQ                              ← 기존 유지
⑥ CTA                              ← 기존 유지
```

---

### ① 히어로 수정
- `"3,500만 국내 밴드 사용자"` → `"글로벌 가입자 1억 3,000만명"`
- 우측 특징 카드 첫 항목 수정

 페이스북·틱톡보다 높은 국내 이용자를 보유한 검증된 플랫폼입니다."
