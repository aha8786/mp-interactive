# 구글 광고 Quality Score 올리는 7가지 방법

Quality Score(품질 점수)는 구글 광고에서 가장 중요한 지표 중 하나입니다. 점수가 높을수록 더 낮은 CPC로 더 높은 순위에 광고가 노출됩니다.

## Quality Score란?

Quality Score는 1~10점으로 구성되며 세 가지 요소로 결정됩니다:

- **예상 클릭률(CTR)**: 광고가 노출될 때 클릭될 확률
- **광고 관련성**: 키워드와 광고문구의 연관성
- **랜딩페이지 경험**: 사용자가 랜딩페이지에서 겪는 경험의 질

## 7가지 개선 방법

### 1. 타이트한 광고 그룹 구성

광고 그룹당 키워드를 5~10개 이하로 제한하고, 테마가 비슷한 키워드끼리 묶으세요.

### 2. 광고문구에 키워드 포함

제목과 설명에 타겟 키워드를 자연스럽게 포함하면 관련성이 높아집니다.

```
✅ 좋은 예: "서울 피부과 | 여드름 치료 전문 | 당일 예약 가능"
❌ 나쁜 예: "최고의 클리닉 | 다양한 치료 | 전문 의료진"
```

### 3. 반응형 검색 광고(RSA) 활용

최소 8~10개의 제목과 4개의 설명을 입력하면 구글이 자동으로 최적 조합을 찾아줍니다.

### 4. 랜딩페이지 최적화

> 광고를 클릭한 사용자가 광고 내용과 관련 있는 페이지로 이동해야 합니다.
> 일반 홈페이지 대신 광고 전용 랜딩페이지를 만드세요.

랜딩페이지 체크리스트:
- 페이지 로드 속도 3초 이하
- 모바일 최적화
- 명확한 CTA (전화번호, 문의 버튼)
- 광고 메시지와 일치하는 콘텐츠

### 5. 광고 확장(Extensions) 활용

사이트링크, 전화번호, 위치, 구조화된 스니펫 등을 모두 설정하면 예상 CTR이 상승합니다.

### 6. 네거티브 키워드 관리

관련 없는 검색어에 노출되면 CTR이 하락합니다. 검색어 보고서를 주기적으로 확인하고 불필요한 키워드를 제외하세요.

### 7. 광고 스케줄링 최적화

업종별 고객이 가장 많이 활동하는 시간대에 입찰가를 높이고, 비활성 시간에는 낮추세요.

## Quality Score와 CPC의 관계

| Quality Score | CPC 대비 영향 |
|---------------|---------------|
| 10점 | -50% |
| 9점 | -44% |
| 8점 | -37% |
| 7점 | -29% |
| 6점 | -14% |
| 5점 | 기준 |
| 4점 | +22% |
| 3점 | +67% |
| 2점 | +150% |
| 1점 | +400% |

점수가 5점 기준에서 1점 높아질 때마다 비용이 절감됩니다.

## 마치며

Quality Score 개선은 단기간에 이루어지지 않습니다. 꾸준한 테스트와 최적화가 필요하며, 전문 운영팀이 있다면 더 빠른 성과를 얻을 수 있습니다.
