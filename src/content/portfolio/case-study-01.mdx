# 뷰티 쇼핑몰 네이버 광고 최적화 사례

## 클라이언트 소개

국내 스킨케어 전문 쇼핑몰로, 자체 브랜드 제품과 편집샵 형태의 상품을 판매합니다.

## 문제점 진단

- 기존 네이버 검색광고 ROAS 150% (목표치의 절반)
- 품질지수(QI) 평균 3점으로 CPC 과다 지출
- 광고 소재 단일화로 클릭률 저조 (CTR 0.8%)
- 전환 추적 미설정으로 데이터 부재

## 전략 수립

1. **키워드 재편성**: 500개 → 150개 핵심 키워드로 집중
2. **광고 그룹 재구성**: 제품 카테고리별 세분화
3. **품질지수 개선**: 광고문구와 랜딩페이지 일관성 확보
4. **A/B 테스트**: 광고 소재 3세트 병행 운영

## 실행 과정

### 1단계: 키워드 분석 (1~2주)

기존 키워드 성과 분석 후 전환이 없는 키워드 350개 제거. 전환율 상위 150개 키워드에 집중.

### 2단계: 광고 소재 개선 (3~4주)

제품명 포함 + 혜택 강조 + 긴급성 표현 세 가지 패턴으로 소재 제작.

### 3단계: 랜딩페이지 최적화 (병행)

모바일 구매 프로세스 단축 (4단계 → 2단계), 상품 상세 페이지 로딩 속도 개선.

## 결과

| 지표 | 이전 | 이후 | 변화 |
|------|------|------|------|
| ROAS | 150% | 430% | +287% |
| CTR | 0.8% | 2.1% | +263% |
| 전환율 | 1.2% | 4.6% | +283% |
| 평균 CPC | 1,200원 | 780원 | -35% |
| 월 전환 수 | 45건 | 172건 | +282% |
